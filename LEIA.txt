Sistema desenvolvido para o tribunal de justiça do RN para listagem dos editais de seleção de 
estagiários usando as seguintes tecnologias:

-HTML
-CSS
-Jasvascript
-PHP
-CodeIgniter
-Jquery
-Bootstrap
-MySQL

Para executar a aplicação basta executar a query do arquivo "estagio2.sql" e acessar o projeto via 
localhost, para acessar o painel de configuração para operações de CRUD basta inserir "/setup" no final
da URL e logar no sistema.
